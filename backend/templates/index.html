Quick API Commands

Copy and run these in terminal:

Check Server Status:

curl http://localhost:5000/api/status

Get Current Rules:

curl http://localhost:5000/api/rules/current

Add New Blocked Domain:

curl -X POST http://localhost:5000/api/rules/add -H "Content-Type: application/json" -d '{"domain": "malware.com", "category": "blocked"}'
